# Short Summary
This is a tennis robot project by https://github.com/SIN-Amb and my. We built a robot that cleans the court and collects the balls fully automated. The robot is controlled by an Mobile or Windows app.

This project is supported by Blueprint Hackclub, thank you very much

If you are interessted in building something and money is the limit check out: https://blueprint.hackclub.com

# Key Features
* Robot that collects all balls automatically
* Robot that cleans the tennis court by attaching the cleaning net at a hook
* Controlled by a mobile application
* Clean design
