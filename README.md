# Short Summary
This is a tennis robot project by https://github.com/valerypiot and https://github.com/SIN-Amb. We built a robot that cleans the court and collects the balls fully automated. The robot is controlled by an Mobile or Windows app.

# Motivation
We did this project because every tennis player knows that ball collecting is boring and uninteresting. We wanted to solve and combine it with another problem cleaning the court. Also a problem that no tennis player likes. So we came up with this idea of building a tennis robot that does all those things for us. Our goals is to solve this problem and ship it to other as userfriendly and open as possible.
This project is funded by Blueprint Hackclub, shout out to them!
https://blueprint.hackclub.com

# Key Features
* Robot that collects all balls automatically
* Robot that cleans the tennis court by attaching the cleaning net at a hook
* Controlled by a mobile application
* Clean design

# How to use it
* Print all the 3d parts in the /hardware/3d_printing_parts/
* Download the apk
  * If you are Apple User download flutter and clone our project and export. We dont have a mac and so could not export the app as .ipa.
* Set up your Orange Pi 5 Pro -> Download ubuntu and in ubuntu download ros2 and clone or repo hardware/tennisrobot

# Impact
This is one of the first tennis robot that is free and open source for use. Now everyone who is enjoying tennis but not collecting the balls and cleaining the court can build a tennis robot.
